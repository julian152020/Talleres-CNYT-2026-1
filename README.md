# Librería de Números Complejos en Python

Este proyecto implementa una librería en Python para realizar operaciones básicas entre números complejos.  
Los números complejos se representan mediante **tuplas**, usando coordenadas cartesianas y polares.

## Operaciones implementadas

1. Suma 
2. Resta  
3. Producto
4. División
5. Módulo  
6. Conjugado  
7. Cartesiana a polar  
8. Polar a cartesiana  
9. Fase

## Pruebas automáticas

- Se implementan dos pruebas por cada función
- Las pruebas validan casos normales y casos borde
- Todas las pruebas se ejecutan correctamente
